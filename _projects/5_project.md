Project page: [GitHub](https://github.com/fitushar/NoMAISI)

NodMAISI is a generative framework using flow-based diffusion and ControlNet conditioning to create realistic synthetic CT images with AI-generated lung nodules for data augmentation. The framework enables controlled synthesis of diverse nodule presentations, supporting scalable training data generation for lung cancer screening AI systems.
